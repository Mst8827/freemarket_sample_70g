# README

## usersテーブル ※gem deviseを利用する
|Column|Type|Options|
|------|----|-------|
|nickname|string|null: false|
|email|string|null: false, unique: true|  ※devise導入時に自動生成
|password|string|null: false|             ※devise導入時に自動生成

### Association
- has_many :items
- has_one :profile
- has_many :comments
- has_one :credit_card
- has_one :location
